== Installation ==
Starting with the RIESTERWP Core plugin consists of just two steps: installing and setting up the plugin.

### INSTALL RWP MANUALLY

1. Upload the ‘rwp’ folder to the /wp-content/plugins/ directory;
2. Activate the RIESTERWP Core plugin through the ‘Plugins’ menu in WordPress;
